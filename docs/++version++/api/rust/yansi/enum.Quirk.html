Enum representing a yansi quirk.

See the crate level docs for details.

Variantsยง

ยง

Mask

Mask: omit when painting is disabled.

Typically applied via the mask() builder method.

See the crate level docs for details.

ยง

Wrap

Wrap the value: replace resets with the wrapped styling.

Typically applied via the wrap() builder method.

See the crate level docs for details.

ยง

Linger

Linger: do not reset the style after it is applied.

Typically applied via the linger() builder method.

See the crate level docs for details.

ยง

Clear

๐Deprecated since 1.0.1: renamed to Resetting due to builder method conflicts with Vec::clear(). Quirk::Clear will be removed in a future release.

Deprecated: Use Quirk::Resetting instead.

ยง

Resetting

Always reset styling afterwards, even if no actual styling was applied.

Overrides the Linger quirk if present.

Typically applied via the resetting() builder method.

ยง

Bright

Brighten the foreground color if it is not already bright.

Typically applied via the bright() builder method.

See the crate level docs for details.

ยง

OnBright

Brighten the background color if it is not already bright.

Typically applied via the on_bright() builder method.

See the crate level docs for details.
